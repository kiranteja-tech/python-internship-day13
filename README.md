# Web Scraping Using BeautifulSoup

## Objective
To scrape data from a legal and scrape-friendly website using Python.

## Tools Used
- Python
- BeautifulSoup
- requests

## Description
The script fetches quotes from https://quotes.toscrape.com, extracts:
- Quote text
- Author name
- Tags

The extracted data is stored in a CSV file.

## Output
- quotes.csv

## Ethical Considerations
- Scraped only a public demo website
- No login or private data accessed
- Followed robots.txt guidelines
